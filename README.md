# Cpp_Concurrency
Based on **C++ Concurrency in Action** (2nd Edition) by Anthony Williams.

This repository stores chapter notes and code examples followed along with the book.

## Structure

```
chapters/
  ch01/  - Hello, world of concurrency in C++
  ch02/  - Managing threads
  ch03/  - Sharing data between threads
  ch04/  - Synchronizing concurrent operations
  ch05/  - The C++ memory model and operations on atomic types
  ch06/  - Designing lock-based concurrent data structures
  ch07/  - Designing lock-free concurrent data structures
  ch08/  - Designing concurrent code
  ch09/  - Advanced thread management
  ch10/  - Parallel algorithms
  ch11/  - Testing and debugging multithreaded applications
```

Each chapter folder contains:
- `notes.md` – reading notes and key takeaways
- `src/`     – code examples and exercises
